# SFML 3.0 C++ Game Framework

Welcome to the Abertay University CMP105 Game Framework, updated for the 2025 academic year. This simple framework is built using **C++17** and **SFML 3.0**. It is designed to be a learning tool for first-year games programming students.

This repository can be used as a starting point for lab work, group projects, or as a demonstration tool for workshops and school visits.
